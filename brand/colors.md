# HWA Packages – Color Scheme

> Colors extracted from the official HWA Packages logo.

## Primary (Brand Blue)
| Name         | Hex       | Usage                          |
|--------------|-----------|--------------------------------|
| Navy         | `#071a3e` | Hero deep background           |
| Blue Dark    | `#0f2d6b` | Primary dark, category strip   |
| Royal Blue   | `#1a4fa0` | Primary buttons, accents       |
| Mid Blue     | `#2b6fd4` | Hover states                   |
| Sky Blue     | `#1e90d4` | Swoosh accent, highlights      |
| Blue Light   | `#e8f0ff` | Card backgrounds, section tints|

## Supporting Colors (Product Categories)
| Name          | Hex       | Usage                          |
|---------------|-----------|--------------------------------|
| Orange        | `#c2620a` | Paper bags category            |
| Orange Light  | `#fff3e8` | Orange card backgrounds        |
| Purple        | `#6d28d9` | Press items category           |
| Purple Light  | `#f3eeff` | Purple card backgrounds        |
| Teal          | `#0d6e6e` | Customize items category       |
| Teal Light    | `#e6f7f7` | Teal card backgrounds          |

## Neutrals
| Name       | Hex       | Usage                          |
|------------|-----------|--------------------------------|
| White      | `#ffffff` | Page background                |
| Off-White  | `#f5f7fc` | Section backgrounds            |
| Gray 200   | `#e5e7eb` | Borders, dividers              |
| Gray 400   | `#9ca3af` | Placeholder text               |
| Gray 600   | `#4b5563` | Body text (secondary)          |
| Dark Navy  | `#1a2540` | Footer background              |

## Hero Gradient

```

## Mission Banner Gradient

background: linear-gradient(135deg, #071a3e 0%, #1a4fa0 100%);
```
