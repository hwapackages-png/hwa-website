W-Cut Bag — Costing Sheet

Fabric Area = (2 x Width + 2) x Height
Bag Weight (g) = (Area x GSM) / 1550 / (1 + Bonus% / 100)
Pcs per Kg = 1000 / Bag Weight
Unit Price = (Kg Price + Print/kg) x Bag Weight / 1000

Parameters

Base Price (Rs/kg)
Bonus %Extra pieces due to W-cut shape removing material

Stitch Bag — Costing Sheet

Main Palla (g) = (2H + G + 4) x W x GSM / 1550
Side Gussets (g) = 2 x G x (H + 3) x GSM / 1550
Handles (g) = 2 x 2" x HandleLen x GSM / 1550
Maghzi (g) = MaghziW x (2H + G + 5) x GSM / 1550
Total Weight = Main + Gussets + Handles + Maghzi
Unit Price = (Kg Price + Print/kg) x Weight / 1000 + Stitch Cost

Parameters

Base Price (Rs/kg)
Handle Length (inches)Length of each handle strip (2" wide)
Maghzi Width (inches)Width of binding/edge strip
Stitch Cost (Rs/piece)Stitching labour cost per bag

Pricing Formula Overview

Unit Price = (Kg Price + Print Addon/kg) x Bag Weight (kg)
Final Price = Unit Price x Bulk Discount
Total = Final Price x Quantity

Bag Weight formula varies per bag type — see each tab for the specific calculation.

Printing Add-ons (Rs. per kg)

Multiplied by bag weight (kg) and added to the unit price.

Print TypeAdd-on (Rs.)

Bulk Discounts

Discount applied based on quantity. Enter as a decimal (e.g., 0.92 = 8% discount). Checked from highest quantity first.
